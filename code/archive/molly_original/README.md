# Archive — Molly's original analysis scripts

These are the original R scripts written by Molly for the LTH (long-term heating ×
wounding × *Acropora pulchra*) experiment, preserved here unmodified for
provenance and attribution.

They are **not** part of the reproducible pipeline (`code/_run_all.R`). The
current analysis in `code/01–21_*.R` was rebuilt from the raw data in
`data/raw/` and supersedes these for the manuscript, but these scripts are the
original source of the analytical approach and are kept as the record of record.

| File | Response |
|------|----------|
| `LTH_PAM.R` | PAM Fv/Fm photochemical efficiency |
| `LTH_color_card_log.R` | Siebeck D-scale color / pigmentation |
| `LTH_buoyant_weight_growthbytreatment.R` | Buoyant-weight growth by treatment |
| `LTH_physio_characterization_log.R` | Physiological / morphological characterization |
