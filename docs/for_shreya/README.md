# For Shreya — RNA-seq analysis & genet-matching

This folder holds the collaborator brief for **Shreya Banerjee** (UC Davis, Bay lab)
on the LTH project (#17, heat × wound × *A. pulchra*, Mo'orea 2025).

- **`analysis_proposal.md`** — proposed RNA-seq differential-expression analysis,
  the phenotype-anchored hypotheses it should test, and a concrete plan for
  matching the three LTH thickets (A, C, D) to Ross Cunning's genotyped CBASS
  *A. pulchra* genets from Mahana, Mo'orea (Cunning et al. 2024).

The phenotypic side of the project is complete and reproducible (`code/_run_all.R`,
results in `RESULTS.md` and `output/tables/20_master_results.csv`); the RNA-seq is
the open piece this brief is about.
